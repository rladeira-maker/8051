Program compiled with SDCC.

Addresses:
SCL = P1_7 (config.h)
SDA = P1_6 (config.h)
DS3231 = 0xD0 (0x68 I2C for read)
LCD HD44780 = (0x27 I2C for write)

Library to use HD44780 in IIC. The rest of the programs may be incomplete and
not work.
